# Experience Furniture World

A responsive, GitHub Pages-ready website for Furniture World / Super Sales, Ranchi.

## Brand assets included
- Original Furniture World logo preserved without redesign or font replacement
- Actual Furniture World outdoor night photograph used on the first screen

## Files
- `index.html` — website structure and content
- `styles.css` — complete responsive styling
- `script.js` — catalogue filtering, navigation and interactions

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all three files to the repository root.
3. Open **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose the `main` branch and `/root`.
6. Save and wait for the public URL.

## Important before public launch
- Replace the temporary web-hosted reference photographs with owned or licensed brand photographs.
- Replace sample testimonial copy with verified customer feedback.
- Confirm showroom timings before adding them.
- Consider connecting the enquiry actions to a CRM or form service.
- The map and Instagram buttons are already clickable.
